This project analyzes retail transaction data to generate actionable business insights for marketing strategy, customer targeting, and revenue optimization.

The analysis includes data preprocessing, exploratory data analysis, customer segmentation, clustering, cohort analysis, customer lifetime value estimation, and KPI reporting to support data-driven decision making.

---

## Key Objectives
- Understand customer purchasing behavior
- Identify high-value customer segments
- Analyze sales performance across regions, products, and time
- Improve marketing targeting using RFM and clustering
- Estimate customer lifetime value (CLV)
- Track retention using cohort analysis
